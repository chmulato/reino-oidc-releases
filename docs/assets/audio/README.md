# Áudio — Reino OIDC (opcional)

Para tocar um **som de sino de catedral** ou **fanfarra real** no momento da ativação (coroação), coloque um arquivo de áudio em um destes nomes:

- `sino_catedral.mp3`
- `fanfarra_real.mp3`
- `campana.mp3`

O script `sucesso_ativacao.js` tenta reproduzir o primeiro arquivo que existir. Se nenhum estiver presente, a celebração ocorre sem som (sem erro).
